 Stella OS 物理基準天体暦

【トピック】

天文観測で使われる精密な天体暦を、一般的な占星術暦ではなく物理的な天文学の視点から組み直した「物理基準天体暦」シリーズです。惑星ごとの観測中心による見え方の差を扱い、「天文×占星術」を架橋する天文占星術の基盤を目指したものです。

【アイテム説明】

Stella OS で開発を進めている「物理基準天体暦」シリーズです。

このシリーズでは、地球中心に固定された一般的な天体暦とは異なり、各天体を観測中心として設定し、その天体固有の基準面・春分点をもとに天体位置データを構成しています。

現在公開している各データは、単なる観賞用サンプルではなく、Stella OS における座標系研究・検証・実装の流れの中で制作している正式公開データです。  
ご購入は、各データの利用に加えて、今後の開発・検証・公開継続を支えていただく意味も含みます。

物理基準天体暦は、観測中心ごとに異なる視点を持つため、同じ「天体の動き」でも見え方が変わります。  
Stella OS では、この視点差そのものを扱えることを重視し、惑星ごとの天体暦を順次整備しています。
