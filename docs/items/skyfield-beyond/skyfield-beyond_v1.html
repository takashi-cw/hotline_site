この本は何の本か

本書は、Python の天文計算ライブラリ Skyfield と NASA JPL DE440s（JPL DEシリーズ） を基盤に、
天文・占星術計算システムを一から設計するための技術書です。

ただし、これは単なる Skyfield の使い方本 ではありません。

Skyfield が返してくれるのは、基本的には 天体の座標（数値） です。
そこから先の、

  • ハウス計算
  • アスペクト計算
  • トランジット
  • 暦変換
  • 多重心での観測中心切り替え
  • 惑星固有時間

といった、占星術や暦法に必要な「その先」は、すべて別途設計と実装が必要になります。

本書は、その “Skyfieldの先” を、日本語で体系的にまとめたものです。

こんな方に向いています

  • 占星術ツールを自分で開発したい方
  • Skyfield を使って、その先の実装まで進みたい方
  • 既存の占星術ソフトが返す値の「計算根拠」を知りたい方
  • 天文計算を Python で実践的に学びたい方
  • Swiss Ephemeris に依存しない設計を考えている方

逆に、

  • とにかく結果だけ出ればよい
  • ライブラリが返した値をそのまま使えれば十分
  • 実装根拠や検証工程にはあまり関心がない

という場合は、少し重たく感じるかもしれません。

そう思った方は 「Swiss Ephemeris」 をご利用することをお勧めします。

本書の内容

全体は 全11章 + 付録3つ で構成しています。

第1章　時刻系

JST / UTC / TT / TDB、ΔT補正など。
天体計算の入口にして、最初の落とし穴でもある部分です。

第2章　座標系

J2000.0 と of-date、歳差、トロピカル / サイデリアル / IAU の違いなど。
「黄経」と一言で言っても、何の黄経なのかを確認します。

第3章　天体の運動

軌道要素、逆行、惑星固有時間。
占星術で見ている“見え方”の背後にある物理を扱います。

第4章　観測と座標変換

光行時間補正、光行差、重力偏向など。
「見える位置」と「幾何学的位置」が一致しないことを整理します。

第5章　暦

24節気、農暦、月相。
東洋占星術や暦法に繋がる章です。

第6章　天球と地平

天球座標、地平座標、出没計算。

第7章　恒星と星官

Hipparcos、中国星官。

第8章　ハウスシステム

Placidus / Koch / Equal などの数学的導出と高緯度問題。

第9章　占星術的計算

アスペクト、トランジット、アラビックパーツ、ディグニティ。

第10章　多重心天文学

地球以外の惑星を観測中心とするチャート設計。
本書の中核のひとつです。

第11章　自己検証の設計

外部正解のない計算を、どう検証するか。

付録

  • 付録A　東洋占星術の天文学的基盤
  • 付録B　データ取得・配置ガイド
  • 付録C　バイブコーディング実践ガイド — AI開発時のプロンプト設計

この本の特徴

本書は、単なる理論書でも、単なるサンプルコード集でもありません。

  • 擬似コード による言語非依存の解説
  • 各章に 検証用Pythonスニペット
  • 実装の背景や失敗を含めた 開発回想録コラム
  • Skyfield の使い方そのものではなく、Skyfieldの先の設計 を扱う構成

という形で、理論・実装・検証 をつなぐ本にしています。

なぜこの本を書いたのか

Skyfield を触っていると、
「値は出る。でも、その値は何の値なのか？」
という問題に必ずぶつかります。

たとえば同じ黄経のように見えても、

  • どの時刻系か
  • どの座標系か
  • 視位置か幾何位置か
  • of-date か J2000.0 か

によって意味は変わります。

つまり、値が返ってきたこと と
その値が欲しい定義に対して正しいこと は別です。

私はそこを曖昧にしたくなかったので、
Skyfield の先を自分で設計し、検証し、その過程ごと一冊の本にまとめることにしました。
