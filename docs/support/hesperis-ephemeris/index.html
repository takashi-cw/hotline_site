アプリについて

Hesperis Ephemeris は、JPL（NASA ジェット推進研究所）の天体暦データ（DE440s）を直接読み込み、高精度な惑星位置計算を行う macOS 用天文アプリです。

  • 惑星・月・太陽の黄経・黄緯・距離の計算
  • 複数の座標系（J2000.0・of-date・固有 of-date）に対応
  • エフェメリス TSV 出力
  • 惑星レーンビュー（Science Q）
  • 2D 俯瞰オーラリー

よくある質問

Q. BSP ファイルとは何ですか？

NASA/JPL が公開している天体暦データファイルです。本アプリは DE440s.bsp を使用しており、アプリに同梱されています。インターネット接続は不要です。

Q. 計算結果の精度はどのくらいですか？

JPL DE440s 天体暦に基づく計算を行っており、一般的な用途において十分な精度を持ちます。詳細は著者の関連論文をご参照ください。

Q. 対応している macOS のバージョンは？

macOS 26（Tahoe）以降を推奨します。開発環境: macOS Tahoe 26.5 / Apple M2 / 8GB RAM

Q. データはどこかに送信されますか？

送信されません。すべての計算はデバイス内で完結します。詳細はプライバシーポリシーをご確認ください。
